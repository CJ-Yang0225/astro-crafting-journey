## 程式碼區塊測試

以下是一個 TypeScript 程式碼範例：

```typescript
interface ReadingTimeData {
  text: string;
  minutes: number;
  time: number;
  words: number;
}

function calculateReadingTime(content: string): ReadingTimeData {
  const wordsPerMinute = 200;
  const words = countWords(content);
  const minutes = Math.ceil(words / wordsPerMinute);
  
  return {
    text: `${minutes} 分鐘`,
    minutes,
    time: minutes * 60 * 1000,
    words
  };
}
```

程式碼區塊應該以較慢的速度計算，因為閱讀程式碼需要更多時間理解。
